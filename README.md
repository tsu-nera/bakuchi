bakuchi
===

仮想通貨Botトレード開発

## 概要

仮想通貨Botを開発して億り人になり、武蔵小杉のタワマンに住むためのプロジェクト。

戦略は、とりあえずローリスク・ローリターンのアービトラージ(裁定取引)で実装する。

* [裁定取引 \- Wikipedia](https://ja.wikipedia.org/wiki/%E8%A3%81%E5%AE%9A%E5%8F%96%E5%BC%95)

いまいちうまくいかなければ他の戦略を考えることにする。
